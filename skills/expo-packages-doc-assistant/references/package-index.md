# Expo Package Index

Quick-lookup table: map user terms to local doc folders.

## How to use this file

Scan this table to find the right `packages/<slug>/` folder for the user's request.
Match against the **User terms** column — these cover npm package names, common aliases, and natural-language phrases users tend to type.
The **Size** column (S/M/L) indicates the doc file size — use it to decide whether to read the full file or scan selectively (see SKILL.md "Lookup workflow" step 3).
